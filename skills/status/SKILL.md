# ステータス確認

全案件または特定案件のステータスを確認・表示します。

## 実行手順

### 引数なし（全案件一覧）
Plugin Root 配下の `dashboard/active-projects.md` を読み込み、全案件のステータスを表示:

```
## 案件ステータス一覧（YYYY-MM-DD時点）

### 稼働中案件
| 案件ID | クライアント | 案件名 | Phase | 進捗 | 優先度 | 期限 | 状態 |
|--------|------------|--------|-------|------|--------|------|------|

### パイプライン（商談中）
| 案件ID | クライアント | 概要 | ステータス |
|--------|------------|------|----------|

### 最近完了した案件
| 案件ID | クライアント | 案件名 | 完了日 |
|--------|------------|--------|--------|
```

注意が必要な案件（遅延、ブロック等）があれば強調表示する。

### 案件IDが指定された場合（詳細表示）
`projects/{案件ID}/` の全ファイルを確認し、詳細ステータスを表示:

```
## 案件詳細: {案件ID} - {案件名}

### 基本情報
- クライアント: {名前}
- Phase: {現在のPhase}
- 優先度: {高/中/低}
- 開始日 / 期限: {日付}

### 進捗
{progress.md の内容サマリー}

### タスク状況
- 完了: X件 / 進行中: X件 / 未着手: X件 / ブロック: X件

### 最近の意思決定
{decisions.md の直近3件}

### リスク
{risks.md の未解決リスク}

### 各部署の最新レポート
{reports/ の各レポートのサマリー}
```
